# 入退室記録 運用メモ

## 現在の正本
- iPad 用Webアプリ
- 公開URL: `https://atuide.github.io/attendance-ipad-app/`
- Excel取込ブック
- Excel取込ブック再生成スクリプト

## 補足
- ファイル名や保存場所は今後変わる可能性がある
- このメモでは `役割` を基準に書く
- 実際のファイル名・保存先は、その時点の最新管理場所に従う

## 日常運用
1. iPad で公開URLを開く
2. `記録` タブで担当ボタンを押す
3. `入室 / 退室` は自動切替
4. 必要なら `本日の記録をクリア` で当日分だけ削除する

## 記録ルール
- `0:00-11:59` は自動で `入室`
- `12:00-23:59` は自動で `退室`
- `入室 / 退室` ボタンを押すと、そのモードに `30秒だけ` 手動切替
- `10秒以内` の同一担当・同一区分の連打は無視

## 画面仕様
### 記録
- 左: 担当ボタン
- 右上: 本日の入退室
- 右下: 本日の最新記録

### 月次
- 選択した月を一覧表示
- `選択した月をCSV出力`
- `選択年度をCSV出力`
- `月次CSVをメールで送る`
- `年度CSVをメールで送る`

### 設定
- 名前変更
- 有効/無効
- メール送付先
- `設定、記録データを保存`
- `保存データを読込`
- `初期化`

## データ保存
- iPad のブラウザ内保存を使用
- 保存対象:
  - 名前設定
  - 有効/無効
  - 入退室ログ
  - UI設定の一部

## CSV出力仕様
### 月次CSV
- ファイル名: `PMS006-YYYY-MM.csv`
- 列:
  - `date`
  - `year_month`
  - `first_in_name`
  - `first_in_time`
  - `last_out_name`
  - `last_out_time`

### 年度CSV
- ファイル名: `PMS006-YYYY年度.csv`
- 年度は `4月〜翌3月`
- 列は月次CSVと同じ

## メール送信の現在仕様
### 月次CSVをメールで送る
- 共有シートで `月次xlsx` を添付共有する
- 送付先が設定されていれば、共有本文は `送付先: メールアドレス`

### 年度CSVをメールで送る
- 共有シートで `年度csv` を添付共有する
- 送付先が設定されていれば、共有本文は `送付先: メールアドレス`

### 注意
- iPad Safari では `添付自動` と `宛先自動入力` の両立はしていない
- 現在は `添付自動` を優先
- 宛先は本文を見てコピーして使う運用

## Excel取込運用
1. iPad から `月次CSV` または `年度CSV` を出力
2. PC で Excel取込ブック を開く
3. `CSV読込` を押す
4. CSV を選択する

## Excel取込マクロの現在仕様
- 月次CSV:
  - 対象月の1シートに転記
- 年度CSV:
  - `YYYY-MM` の月別シートを自動作成
  - 各月ごとに転記

## クリア機能
- `本日の記録をクリア`
- 今日の日付のログだけ削除
- 影響範囲:
  - 本日の入退室
  - 本日の最新記録
  - 月次一覧の当日分

## 注意点
- GitHub Pages の反映には少し時間がかかる
- 見た目を直した直後は、ページ再読み込みが必要
- Excel取込ブックを開いたままだと再生成に失敗することがある

## 変更時の原則
- `attendance_ipad_app` の変更は GitHub に push する
- Excel 側仕様を変えたら Excel取込ブック を再生成する
